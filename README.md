## Getting Started
### Installation
- Clone this repo:
```bash
git clone git@github.com:littlebaba/doubanbooks.git
cd doubanbooks
```
- Install dependencies
    - For pip users,please type the command `pip install -r requirements.txt`.
- Run
    - enter main.py run the main funtion
